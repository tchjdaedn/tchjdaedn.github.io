Summary: Latitude Vs. X

This site was created to illustrate various weather trends in cities around the world based on their latitude. It is divided into various sections, the basis of which is this homepage which can be accessed at any time by clicking the "Latitude" link in the upper left. The additional sections can be accessed through the links in the upper right (or drop-down menmu if you are using a low-res platform such as a mobile device).


The second set of pages are the Visualization descriptions. These are accessible through the individual links in the "Plots" drop-down menu. Each page focuses on a single data visualization and provides a bit more in-depth detail.

The next page is the Comparisons page. This contains side-by-side versions of the visualizations for easier comparison and contrast. This page does not contain any further text descriptions.

The final page is the raw data set used to create the visualizations. While this data was originally assembled in a comma-separated-values file, it was loaded into Microsoft Excel and converted into an html table. The date stamp was not converted in the process, and so does not have a standard format.
